# SiliconFlow Balance Assistant

## Introduction
A beautiful, modern web tool to quickly check your SiliconFlow API account balance and status.

## Features
- **Real-time Check**: Directly calls SiliconFlow API to get the latest balance.
- **Modern Design**: Features dark mode, glassmorphism, and dynamic background animations.
- **Privacy First**: Your API Key is used locally and sent directly to the official SiliconFlow endpoint. It is never sent to any third-party server.
- **English Interface**: Clean and professional English UI.

## How to Use
1. Open `index.html`.
2. Paste your SiliconFlow API Key (`sk-...`) into the input field.
3. Click the "Check Balance" button.
4. View your balance, account status, and user info.

## Tech Stack
- HTML5
- CSS3 (Vanilla CSS, no dependencies)
- Vanilla JavaScript

## Structure
- `index.html`: Main page structure
- `style.css`: Styles
- `script.js`: Logic
